## 3.2 程序编码

### 3.2.2 代码示例

Q：如何让gcc对单个C语言文件产生汇编代码即.s文件？

A：在命令行上使用“-S”选项。

```shell
$ gcc -Og -S mstore.c
```

Q：如何让gcc对单个C语言文件产生目标代码文件即.o文件？

A：在命令行上使用“-c”选项。

```shell
$ gcc -Og -c mstore.c
```

Q：如何理解目标代码文件即.o文件？

A：.o文件是二进制格式的，无法直接查看。

结论：机器执行的程序只是一个字节序列，它是对一些列指令的编码。机器对产生这些指令的源代码几乎一无所知。
